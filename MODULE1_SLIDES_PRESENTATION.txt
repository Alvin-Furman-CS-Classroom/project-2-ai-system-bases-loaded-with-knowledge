WHAT IT DOES
───────────────────────────────────────────────────────────────────────────────
• Predicts offensive performance scores (0-100) for Braves batters vs MLB pitchers
• Uses first-order logic rules with quantifiers (∀, ∃) to evaluate matchups
• Produces scores that feed into Module 3 (CSP), Module 4 (Search), Module 5 (Planning)

INPUTS
───────────────────────────────────────────────────────────────────────────────
BATTERS (26 Braves players):
  • BA, OBP, SLG, HR, RBI, K, handedness (L/R/S)

PITCHERS (157 MLB pitchers):
  • ERA, WHIP, K rate, walk rate, handedness (LHP/RHP)

OUTPUTS
───────────────────────────────────────────────────────────────────────────────
• Performance scores (0-100) for each batter-pitcher matchup
• Format: {batter_name: score} or {batter_name: {pitcher_name: score}}

KEY FEATURES
───────────────────────────────────────────────────────────────────────────────
✓ Realistic 2025 MLB statistics from Baseball-Reference
✓ 26 Braves batters aligned with Module 2 defensive data
✓ 157 unique MLB pitchers (2025 season)
✓ Multiple analysis modes: single pitcher, single batter, or full matrix

QUANTIFIERS
───────────────────────────────────────────────────────────────────────────────
UNIVERSAL (∀): "For all batters, if OBP > 0.350 and pitcher walk rate > 0.10,
               then increase score"

EXISTENTIAL (∃): "There exists a batter such that SLG > 0.500 and pitcher
                  ERA > 4.00, then increase score"

EXAMPLE RULES
───────────────────────────────────────────────────────────────────────────────
┌──────────────────────┬──────────────────────────────┬──────────────────────┐
│ Rule Type            │ Condition                    │ Adjustment           │
├──────────────────────┼──────────────────────────────┼──────────────────────┤
│ Handedness           │ Same-handed (L vs LHP)        │ -15.0 penalty        │
│ OBP Advantage        │ OBP > 0.350, walk rate > 0.10│ +8.0 bonus          │
│ Power Matchup        │ SLG > 0.500, ERA > 4.00      │ +10.0 bonus          │

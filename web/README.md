# Module 4 Web UI

This folder contains generated web dashboard assets for Module 4.

## Generate dashboard

From repository root:

```bash
PYTHONPATH=src python3 demos/demo_module4_web_ui.py
```

## View dashboard

Open:

- `web/module4_dashboard.html`

Or serve the folder locally:

```bash
cd web
python3 -m http.server 8000
```

# Demo Scripts

This folder contains demo scripts that demonstrate how to use the various modules.

## Running the Demos

All demos should be run from the repository root directory:

```bash
# From the repo root
python3 demos/demo_matchup_analysis.py
python3 demos/demo_batter_vs_pitchers.py
python3 demos/demo_matchups_matrix.py
python3 demos/demo_defensive_analysis.py
```

Or with PYTHONPATH set:

```bash
PYTHONPATH=src python3 demos/demo_matchup_analysis.py
```

## Demo Files

- **demo_matchup_analysis.py** - Demonstrates analyzing multiple batters against one pitcher
- **demo_batter_vs_pitchers.py** - Demonstrates analyzing one batter against multiple pitchers
- **demo_matchups_matrix.py** - Demonstrates analyzing multiple batters against multiple pitchers (matrix)
- **demo_defensive_analysis.py** - Demonstrates Module 2 defensive performance analysis
